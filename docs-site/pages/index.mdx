# Welcome to Pariflow Documentation

Pariflow is a production-grade prediction market platform where users trade on future outcomes. This documentation covers everything you need to know about using, developing with, and administering the platform.

## What is Pariflow?

Pariflow is a sophisticated prediction market platform featuring:

- **Hybrid Trading System**: Combines AMM (Automated Market Maker) with traditional order book functionality
- **Real-time Trading**: WebSocket-powered live price and order book updates
- **Risk Management**: Automated hedging system with Polymarket integration
- **Sports Integration**: Real-time sports betting markets
- **Institutional APIs**: Professional-grade market data APIs for algorithmic trading
